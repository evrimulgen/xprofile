xprofile
========

Usage
```
./xprofile.rb [option(s)] [filename(s)]
Options:
-p   - Profile Android .apk       |  --open [apk-filepath]
-l   - Use lexical analyzer       |  --lex [src-dir]
-s   - Decompile .apk to source   |  --apk [apk-filepath]

Clean project with 'rake clean'

```

Libraries
```
GATOR
dex2jar
jad
```
